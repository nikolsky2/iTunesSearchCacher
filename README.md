# iTunes Search Cacher
Local caching of preview audio m4a files into external storage using Core Data framework

Using public iTunes search api: https://itunes.apple.com/search?

Description is here https://affiliate.itunes.apple.com/resources/documentation/itunes-store-web-service-search-api/
